# Notes
- The current route name output is not a friendly screen name because `MaterialPageRoute` has no `RouteSettings.name`.
- To get actual screen names, push named routes or supply `RouteSettings(name: '/screenName')` when creating routes.
- If you want this file to capture future logs automatically, run the app and redirect output:
  `flutter run | tee slow_frame_route_logs.txt`
